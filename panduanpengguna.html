Start

Di bagian ini Anda dapat mengikuti langkah-langkah untuk menghubungkan bot dengan cepat ke akun Exchange Anda.

Instalasi

Ikuti langkah-langkah di bawah ini untuk menyelesaikan penyiapan Anda di bot:

  1. Binance: Cara Membuat API
    **Pastikan Anda telah memilih "Enable Spot & Margin Trading". Jika tidak, sistem tidak dapat melakukan pemesanan untuk akun Anda.
  2. Buka akun Telegram Anda dan buka bot perdagangan. Ketik /start jika Anda tidak melihat menu.
  3. Klik Menu Exchanges → Binance → 🔴 Set API Key
    To set your API Key, type /setApiKey -exchange- -apiKey-
    /setApiKey Binance vsXhFh4cbd245...
    If you get the response "Succesful.", go to next step.
  4. Click Exchanges → Binance → 🔴 Set API Secret
    To set your API Secret, type /setApiSecret -exchange- -apiSecret-
    /setApiKey Binance qv26cAVXg326c...
    If you get the response "Succesful.", go to next step.
  5. Click Exchanges → Binance → 🟢 Check API Valid
    If you get the response "API credentials are valid.", go to next step.
  6. Great! You are almost done.
    Click Exchanges → Binance → ✔️Set Connection
  7. Sekarang bot berhasil terhubung ke akun Binance Anda. Saatnya untuk mengonfigurasi Pengaturan Exchange Anda.
    Saatnya untuk mengonfigurasi Pengaturan Exchange Anda.

Exchange Settings

Di bagian ini Anda dapat memeriksa bagaimana Anda dapat melakukan setting pada menu Exchange Settings.

Maximum Signal

Untuk menggunakan anggaran Anda secara efektif dan menerapkan rata-rata biaya dolar, Anda harus menyesuaikan jumlah maksimum sinyal yang akan Anda perdagangkan pada saat yang sama.

Main Menu → Exchange → Binance → Max Signal → Klik berapa banyak sinyal yang ingin Anda perdagangkan secara bersamaan.

Re-buy

Artinya jika Anda berada dalam sinyal dan harga turun, bot akan secara otomatis membeli kembali koin untuk menurunkan harga beli rata-rata Anda. Rata-rata biaya dolar adalah metode yang sangat membantu untuk keluar dari sinyal sedini mungkin. Namun, perlu dicatat bahwa ini mungkin juga membuat Anda berada dalam posisi dengan kerugian besar. Putuskan baik-baik berapa kali akan buyback lagi.

Main Menu → Exchange → Binance → Re-buy → Klik berapa kali Anda ingin membeli kembali.

%Re-buy

Untuk menerapkan Dollar-Cost Averaging, Anda harus menentukan jarak agar bot membeli lagi untuk Anda. Mari kita periksa situasi ini Anda membeli kembali % = 3% Anda membeli sinyal saat 10000 satoshi, artinya pembelian ulang pertama Anda adalah saat harga 9700 satoshi. Pembelian ulang kedua Anda akan terjadi saat harga 9700*0,97=9409 satoshi Pembelian ulang ketiga Anda adalah saat harga 9409*0.97=9127 satoshi

Main Menu → Exchange → Binance → %Re-buy → Klik berapa persentase yang ingin Anda tetapkan.

Stop Loss

Stop-loss dapat didefinisikan sebagai pesanan di muka untuk menjual aset ketika mencapai titik harga tertentu. Ini digunakan untuk membatasi kerugian atau keuntungan dalam perdagangan. Dengan menempatkan order stop-loss, investor menginstruksikan broker/agen untuk menjual sekuritas ketika mencapai batas harga yang telah ditentukan sebelumnya.

Main Menu → Exchange → Binance → Stop Loss → Klik berapa persentase yang ingin Anda tetapkan.
Note Jika Anda tidak ingin mengatur stop-loss, klik ❌

Trail Stop

Trail stop terdiri dari 2 komponen. Yang pertama adalah di mana untuk mulai melacak keuntungan dan yang kedua adalah dengan seberapa jauh jarak untuk mengikutinya. Misalnya, Anda mungkin ingin berhenti pada tingkat keuntungan 1% ketika Anda berada dalam posisi dengan keuntungan 2%. Berkat fitur pelacakan keuntungan, saat tingkat keuntungan Anda meningkat, level stop Anda akan naik dan keuntungan Anda akan dimaksimalkan. Jika profit Anda 3%, stop Anda akan dipindahkan ke 2%. Saat harga terus naik, level stop Anda akan meningkat, dan saat harga mulai turun kembali, level stop Anda akan terpicu dan Anda akan keluar dari posisi. Untuk mengatur trail stop,
cukup ketik /setTrailStop -exchange- -startLevel- -distance-

For example: /setTrailStop Binance 2 0.5.

Cancel Order Time

Dalam beberapa kasus, sinyal dapat diselesaikan sebelum pesanan yang kami buat dipenuhi, dan pesanan mungkin tetap terbuka tanpa alasan. Oleh karena itu, adalah kepentingan terbaik kami untuk membatalkan pesanan kami secara otomatis setelah jangka waktu tertentu.

Main Menu → Exchange → Binance → Cancel Order → Klik berapa menit Anda ingin menunggu sebelum membatalkan pesanan.

Initial Margin

Ketika sinyal baru diterbitkan dalam algoritme yang Anda ikuti, Anda mungkin ingin memasukkan lebih awal atau lebih lambat daripada harga yang dipublikasikan.

atau contoh, ketika sinyal disiarkan dengan harga 1000 USDT,

  • Jika margin pesanan awal Anda adalah -1%, pesanan dikirim untuk memasukkan sinyal 1% lebih rendah, yaitu 990 USDT.
  • Jika -1,5%, pesanan dikirim sebagai 985 USDT.

Jika Anda ingin segera memasukkan sinyal, Anda dapat mengaktifkan pembelian pasar. Dengan cara ini, pembelian akan segera dilakukan.

Main Menu → Exchange → Binance → Initial Margin → Klik berapa persentase yang ingin Anda tetapkan.
Note Jika Anda tidak ingin mengatur biarkan saja Initial Margin:0%

Per Buy Limits

Anda perlu menetapkan nilai per beli agar bot dapat melakukan pemesanan saat dalam sinyal. Nilai ini sangat penting dan tergantung pada selera risiko pengguna. Disarankan untuk menetapkan 1/20 dari total saldo Anda sebagai nilai optimal. Untuk mengatur jumlah pembelian Anda per perdagangan,
cukup tulis /setPerBuy -Binance- -Asset- -amount-

Contoh cukup tulis /setPerBuy Binance USDT 40

Blacklist

Blacklist adalah koin atau token Cryptocurrency yang di hindarkan oleh robot untuk tidak di perdagangkan.
artinya apabila ada signal baru masuk maka tidak dieksekusi oleh robot.
Main Menu → Exchange → Binance → Blacklist

/addBlackList Binance XRPUSDT Untuk menambahkan koin ke daftar Blacklist
Note /removeBlacList Untuk menghapus koin dari daftar Blacklist Anda

Balance

Untuk melihat sisa modal kita dalam bentuk Usdt dan Currency lain .

Main Menu → Exchange → Binance → Balance
